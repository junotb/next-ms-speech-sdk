# Next MS Speech SDK - Speech To Text
[Next.js](https://nextjs.org/)로 제작한 MS Speech SDK를 사용한 STT(Speech To Text) 샘플.

## 시작하기

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
